🏦 Bank Management System using Dart OOP

A simple console-based Bank Management System built with Dart using Object-Oriented Programming (OOP) concepts.

//Features

- Create Bank Account
- Deposit Money
- Withdraw Money
- Check Balance
- Display Account details
  

//Technologies Used

- Dart
- Object-Oriented Programming (OOP)

 OOP Concepts Used

- Class
- Object
- Constructor
- Encapsulation
- Methods
